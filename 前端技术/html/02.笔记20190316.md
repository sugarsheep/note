1. html如何引用图片

   通过img标签引用图片，该标签有4个常用属性

   ```html
   <!--
   src:图片路径，一般使用相对路径，相对于当前页面，../可以返回上一级目录，多个../即可返回多级目录
   alt:图片不显示时的描述信息，也可用于搜索引擎检索图片
   width：图片宽度
   height：图片高度
   width和height只设置其中一个值时，图片会等比例缩放，单位一般为px
   -->
   <img src="a.jpg" alt="竹子" width="20px" height="20px"/>
   ```

2. 常用的图片格式有哪些？分别有什么特点？

   ```
   JPEG(JPG):支持的颜色较多，可压缩，不支持透明
   GIF:支持的颜色少，支持简单的透明（如直线透明），支持动态图，在颜色单一或需要动态图时使用
   PNG：支持颜色多，支持复杂透明
   
   3种图片格式使用原则：
   	效果不一致，选择效果好的
   	效果一致，选择占用内存小的
   ```

3. meta标签的其它作用

   ```html
   	<!-- 1.声明页面的关键字 -->
       <meta name="keywords" content="hello,world"/>
       <!-- 2.声明页面的描述信息 -->
       <meta name="description" content="这是一个页面"/>
       <!-- 3.设置重定向 -->
       <!-- 其中5表示5秒后跳转，后面跟一个分号 -->
       <meta http-equiv="refresh" content="5;url=https://www.baidu.com"/>
   ```

4. xhtml语法

   ```
   1.标签不区分大小写，建议使用小写
   2.注释不能嵌套
   3.标签必须结构严谨，要么成对出现，要么自结束
   4.标签不要交叉嵌套
   5.标签属性必须有值，且带引号（单引号或双引号）
   ```

5. 内联框架iframe

   ```html
   <!-- 用于引入外部页面，但不会被搜索引擎检索，不推荐使用 -->
   <iframe src="a.html" name="tom"></iframe>
   ```

6. 如何使用超链接

   ```html
   	<!-- 超链接使用a标签 #表示刷新当前页面，#标签id可以跳转到指定标签-->
       <a href="#" target="_self">链接</a>
    	<!-- 发送邮件 -->
       <a href="mailto:sugar@qq.com">联系我</a>
   ```

   ```
   target的常用值：
   	_self:在当前页面打开url
   	_blank:在一个新的页面打开url
   	内联框架的name值：在内联框架打开url
   ```
